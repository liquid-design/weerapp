## Let op bij ODbL (Frankrijk)
De Franse departementgeometrie is OpenStreetMap-afgeleid (ODbL). Publiceer je een **gewijzigde**
versie van dat databestand, dan geldt share-alike: die afgeleide database moet ook onder ODbL.
De MIT-licentie op de code raakt dit niet; het betreft alleen het databestand.
